## Слайд 12: Docker Compose

# 🐳 Docker Compose

```yaml
services:
  db:          # PostgreSQL 16
    ports: ["6789:5432"]
    healthcheck: pg_isready
    
  backend:     # Spring Boot
    ports: ["8081:8080"]
    depends_on: [db]
    
  frontend:    # Vue 3 + Nginx
    ports: ["5173:80"]
    depends_on: [backend]
```

**Изолированная сеть + Health checks**
